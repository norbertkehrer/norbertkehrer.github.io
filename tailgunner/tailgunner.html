Tailgunner in JavaScript

Exact port of the original arcade game to JavaScript using the method of static binary translation.

Dan Sunday … Original game program (1979)

Graham Toal … Static binary translation (2004)

Norbert Kehrer … JavaScript code (2019)

 

Keys to play:
Movement … Mouse or cursor keys
Fire … Mouse button or "Ctrl"
Shield … "Space"
Start Game … "1"
Insert Coin … "5", or "6", or "7"

Control with mouse
