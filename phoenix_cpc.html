The Phoenix Emulator for the Amstrad CPC

This is an emulator of Amstar's classic arcade game "Phoenix" from 1980 running on the Amstrad CPC home computer. The CPC, which has a Z80 CPU, can run the original game program from 1980 written in the arcade machine's 8085 machine language. The graphics and sound hardware is emulated on the CPC. Unfortunately, there are still some bugs and the sound is bad, but I will try to improve that ;-)

The program can be downloaded as a CPC "dsk" file here: phoenix.dsk

You can use this file with a CPC emulator or put it on a floppy disk or on tape and play it on the real machine.

The disk image contains two files. Run the file "phoenix" for the 16-color version with lower resolution, or alternatively run the file "phoenixh" for a 4-color version with high resolution.
