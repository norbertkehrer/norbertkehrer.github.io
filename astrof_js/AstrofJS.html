Astro Fighter in JavaScript

Original game by Data East in 1980.

Exact port of the original to JavaScript using the method of static binary translation in March 2015 by Norbert Kehrer.

Read more about it on my homepage at https://norbertkehrer.github.io

Keys to play:
Left Crsr Left, Z, or Y
Right Right or X
Fire Ctrl or M
Start 1 Player 1
Start 2 Players 2
Insert Coin 5, or 6, or 7
Pause Game P
